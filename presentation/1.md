
## Installation
```sh
npm install @mediamonks/channels
```


## Create an instance

Before we can do anything, an instance of `Channels` has to be created. Two parameters are required:

- the **location of the sound files**
- the **file extension** to use

```javascript
const channelsInstance = new Channels({
    soundsPath: 'location/of/your/files',
    soundsExtension: 'mp3',
})
```
> Don't create more than one instance.
