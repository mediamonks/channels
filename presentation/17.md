
# Audio effects

- An effect has to have an `input` and an `output` (of type `AudioNode`)
- Can contain either a single node or a chain of multiple nodes
- Effects can be placed on either a **channel** or the **main output**.

<div align="center"><img src="https://github.com/petervdn/channels/blob/develop/assets/overview-diagram.png?raw=true" /></div>
<div align="center"><img src="https://github.com/petervdn/channels/blob/develop/assets/volume-diagram.png?raw=true"/></div>


```javascript
const filter = audioContext.createBiquadFilter();
const myEffect = {
    input: filter,
    output: filter,
}
// setting an effect on the main output 
const channelsInstance = new Channels({
    soundsExtension,
    soundsPath,
    mainEffects: myEffect,
});
